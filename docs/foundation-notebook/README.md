# Foundation Notebook

Version: Foundation Edition v0.0.1 (Draft)

Status: Institutional Memory (Non-Authoritative)

---

# Purpose

The Foundation Notebook preserves reasoning behind important discoveries while Life School is still learning.

It answers one recurring institutional question:

Why did we think this?

---

# Constitutional Position

The Foundation Notebook does not create constitutional authority.

It is not publication, doctrine, or policy.

It is a reasoning memory layer that supports future examination.

If notebook content conflicts with constitutional documents, constitutional documents prevail.

---

# How To Use This Family

Use notebook entries to preserve:

- the originating question,
- the conversation trajectory,
- discoveries and reasoning,
- alternatives considered,
- current understanding,
- possible future revisions.

Notebook entries should be concise, honest, and revision-friendly.

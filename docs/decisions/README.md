# Decisions

This family records meaningful institutional decisions and preserves the reasoning behind them.

It supports constitutional traceability and does not replace constitutional authority.

---

# Minimum Decision Entry Fields

- Decision ID
- Date
- Decision summary
- Why this exists
- Alternatives considered
- Constitutional references
- Related dialogues
- Related revisions

Optional:

- Principle references (for example FP-001 when applicable)

---

# Scope Note

Use FP-001 only when a decision directly concerns non-finality, correction, or deepening understanding.

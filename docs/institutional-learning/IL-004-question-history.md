# IL-004
# Question History

Version: Foundation Edition v0.0.1 (Draft)

Status: Operational Draft

---

# Purpose

Question History preserves important institutional questions over time and tracks how answers changed or remained unresolved.

---

# Entry Requirements

Each question entry should include:

- Question ID,
- Date opened,
- Origin context,
- Current working answer,
- Competing alternatives,
- Reasoning status,
- Linked revisions,
- Linked decisions,
- Linked dialogues,
- Date last examined,
- Next examination trigger.

---

# Output

Use templates/QH-ENTRY-TEMPLATE.md.

---

# Logging

Maintain entries in registers/question-history-log.md.

# Institutional Learning

Version: Foundation Edition v0.0.1 (Draft)

Status: Operational Architecture (Non-Constitutional)

---

# Purpose

This family defines how LifeSchool learns as an institution.

It operationalizes one institutional commitment:

LifeSchool should demonstrate the same lifelong learning it encourages in others.

---

# Scope

Institutional Learning documents define process and evidence for:

- Learning Reviews,
- Reflection Reports,
- Institutional Learning,
- Institutional Memory,
- Revision History,
- Question History.

This family preserves reasoning and learning evidence.

It does not create constitutional authority.

---

# Authority

Constitutional authority remains with CA, LS, ST, and derived constitutional specifications.

If this family conflicts with constitutional documents, constitutional documents prevail.

---

# Core Evidence Questions

Every cycle should preserve evidence for:

- What changed?
- Why it changed?
- What was learned?
- What mistakes were made?
- What remains uncertain?
- How understanding deepened?

When these questions explicitly involve principle-level non-finality or correction, related outputs should reference FP-001.

# Public Demonstration

Public transparency can be demonstrated by publishing selected outputs from:

- Learning Review outcomes,
- Reflection Report summaries,
- Revision and Question history excerpts,
- Uncertainty and mistake disclosures.

IL-006 defines publication boundaries and cadence.

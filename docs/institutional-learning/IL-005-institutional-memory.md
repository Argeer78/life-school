# IL-005
# Institutional Memory

Version: Foundation Edition v0.0.1 (Draft)

Status: Operational Draft

---

# Purpose

Institutional Memory consolidates learning evidence so future contributors understand not only conclusions, but also reasoning history.

---

# Memory Inputs

Institutional Memory should consolidate from:

- Learning Reviews,
- Reflection Reports,
- Revision History,
- Question History,
- Decision Records,
- Foundation Notebook entries,
- Relevant archive references.

---

# Consolidation Rules

Consolidation should preserve:

- chronology,
- contradictory phases,
- mistaken assumptions,
- uncertainty that remained unresolved,
- reasoning transitions.

Do not collapse history into a single final narrative.

---

# Output

Publish a periodic memory synthesis note in this family before any major doctrine or strategy revision.

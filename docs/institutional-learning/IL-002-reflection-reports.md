# IL-002
# Reflection Reports

Version: Foundation Edition v0.0.1 (Draft)

Status: Operational Draft

---

# Purpose

Reflection Reports synthesize what LifeSchool learned across multiple reviews and changes over a defined period.

---

# Cadence

Recommended:

- Quarterly institutional reflection,
- Annual long-form reflection.

---

# Required Sections

Each Reflection Report should include:

- Period,
- Major changes,
- Reasoning themes,
- Mistake patterns,
- Uncertainty map,
- Deepened understanding map,
- Open questions,
- Priority re-examinations,
- Public disclosure subset.

---

# Output

The canonical output is one RR entry using templates/RR-TEMPLATE.md.

---

# Logging

Each Reflection Report should be logged in registers/reflection-report-log.md.
